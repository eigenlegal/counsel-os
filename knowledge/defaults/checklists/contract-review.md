# Contract Review Checklist

## Structural Completeness
- [ ] Parties correctly identified with full legal entity names
- [ ] Jurisdiction of incorporation/formation stated for each party
- [ ] Recitals accurately describe the relationship and purpose
- [ ] Effective date clearly stated
- [ ] Definitions section is complete and internally consistent
- [ ] All exhibits, schedules, and order forms are attached and referenced
- [ ] Signature blocks match the identified parties
- [ ] Order of precedence clause addresses conflicts between documents
- [ ] Amendment clause specifies requirements for modifications
- [ ] Notices clause with current addresses and acceptable delivery methods
- [ ] Entire agreement / integration clause present
- [ ] Severability clause included
- [ ] Waiver clause (no waiver unless in writing)
- [ ] Counterparts clause (if applicable)

## Commercial Terms
- [ ] Scope of services or license grant clearly defined
- [ ] Pricing and fee structure matches the proposal/negotiation
- [ ] Payment terms specified (Net 30, Net 60, etc.)
- [ ] Price increase caps and notice requirements defined
- [ ] Invoice dispute process included
- [ ] Late payment interest rate specified and reasonable
- [ ] Tax allocation and gross-up provisions addressed
- [ ] Expenses reimbursement terms (if applicable)
- [ ] Acceptance criteria for deliverables defined
- [ ] Change order process documented

## Term and Termination
- [ ] Initial term length matches business intent
- [ ] Auto-renewal mechanics clearly stated
- [ ] Non-renewal notice period and method specified
- [ ] Termination for cause with reasonable cure period (30 days)
- [ ] Termination for convenience rights (mutual or one-sided)
- [ ] Termination for insolvency/bankruptcy
- [ ] Change of control termination right
- [ ] Post-termination obligations (data return, transition assistance)
- [ ] Survival clause listing provisions that survive termination
- [ ] Early termination fees (if any) are reasonable

## Risk Allocation
- [ ] Limitation of liability — mutual cap proportionate to deal value
- [ ] Consequential damages exclusion with appropriate carve-outs
- [ ] Liability cap carve-outs identified (IP, data breach, confidentiality, willful misconduct)
- [ ] Indemnification obligations are mutual where appropriate
- [ ] Vendor IP infringement indemnity with cure path
- [ ] Data breach indemnification addressed
- [ ] Insurance requirements specified and proportionate
- [ ] Force majeure clause is mutual and reasonable

## Intellectual Property
- [ ] Pre-existing IP ownership retained by each party
- [ ] Customer data ownership confirmed — customer owns its data
- [ ] Custom work product ownership or license clearly assigned
- [ ] License grant for vendor IP embedded in deliverables
- [ ] Open-source disclosure requirements
- [ ] AI/ML training restrictions on customer data
- [ ] Feedback clause (if any) is reasonable
- [ ] No broad assignment of customer IP to vendor

## Data Protection
- [ ] Data Processing Agreement (DPA) included or referenced
- [ ] Data processing limited to customer instructions
- [ ] Sub-processor disclosure and change notification
- [ ] Data breach notification timeline (72 hours or less)
- [ ] Security measures and certifications specified
- [ ] Data localization commitments where required
- [ ] Cross-border transfer mechanisms in place
- [ ] Data return and deletion upon termination
- [ ] Audit rights (direct or third-party)
- [ ] No vendor use of personal data for own purposes

## Confidentiality
- [ ] Mutual confidentiality obligations
- [ ] Reasonable duration (3-5 years, perpetual for trade secrets)
- [ ] Standard exclusions (public knowledge, independent development, prior possession)
- [ ] Need-to-know dissemination limits
- [ ] Return or destruction upon termination
- [ ] Compelled disclosure with notice
- [ ] No problematic residuals clause
- [ ] Injunctive relief acknowledged

## Representations and Warranties
- [ ] Mutual authority and enforceability representations
- [ ] Vendor warrants professional/workmanlike performance
- [ ] Material conformity to documentation warranty
- [ ] Non-infringement warranty
- [ ] Compliance with applicable laws warranty
- [ ] Malware-free and no disabling code warranty
- [ ] Adequate warranty remedies (re-perform, repair, refund)
- [ ] Warranty period defined

## Service Levels (if applicable)
- [ ] Uptime/availability commitment specified (99.9%+)
- [ ] Measurement methodology transparent
- [ ] Service credits for downtime
- [ ] Response and resolution time SLAs by severity tier
- [ ] Planned maintenance windows limited with advance notice
- [ ] Chronic failure termination right
- [ ] Root cause analysis for major incidents
- [ ] Disaster recovery objectives (RPO/RTO)

## Dispute Resolution
- [ ] Governing law specified and appropriate
- [ ] Venue/forum clearly designated
- [ ] Escalation process before litigation/arbitration
- [ ] Injunctive relief carve-out
- [ ] Arbitration rules specified (if arbitration)
- [ ] Prevailing party attorneys' fees (if desired)

## Compliance and Regulatory
- [ ] Anti-corruption / FCPA / UK Bribery Act provisions
- [ ] Export control and sanctions compliance
- [ ] Modern slavery / human trafficking (where applicable)
- [ ] Industry-specific regulatory requirements addressed
- [ ] Accessibility requirements (if applicable)
- [ ] Environmental/sustainability provisions (if applicable)

## Final Checks
- [ ] No blank fields or placeholder text remaining
- [ ] Cross-references are accurate
- [ ] Defined terms are used consistently
- [ ] No internal inconsistencies between sections
- [ ] Exhibits and schedules are complete
- [ ] Signing authority confirmed for both parties
