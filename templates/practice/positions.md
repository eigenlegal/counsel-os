# Practice Positions

This file overrides entries in `knowledge/defaults/positions.md`. For any clause type listed here, your position takes precedence over the market-standard default. If a clause type is NOT listed here, the default position applies.

Format for each position:

```
## [Clause Type]
**Our standard:** [Your organization's standard position]
**We'll accept:** [What flexibility you have — conditions under which you'll deviate]
**We won't accept:** [Hard limits — lines you won't cross]
**Auto-escalate:** [What triggers escalation regardless of other factors]
```

---

## Limitation of Liability
**Our standard:** [your position]

Example:
> Mutual cap at 12 months of fees paid or payable. Uncapped carve-outs for IP
> indemnification, data breach, confidentiality, willful misconduct, and fraud.

**We'll accept:** [flexibility]

Example:
> Cap as low as 6 months for low-value deals (<$50K). Per-claim sub-cap if aggregate
> cap is maintained. Asymmetric carve-outs if we're the customer (vendor has more
> uncapped exposure than we do).

**We won't accept:** [hard limits]

Example:
> Cap below 6 months. No carve-out for data breach. Entirely one-sided cap
> favoring the vendor. Cap based on "amounts paid" rather than "paid or payable"
> for multi-year deals.

**Auto-escalate:** [triggers]

Example:
> Unlimited liability request from either side. Cap below 3 months.
> Vendor refusing any data breach carve-out.
